⛅ WeatherSphere

A production-grade real-time weather application built with React 19, GSAP animations, MUI components, and OpenWeatherMap API integration.

React 19 Vite 8 MUI v9 GSAP 3 OpenWeatherMap API CSS Animations

1. Project Overview

WeatherSphere is a modern, visually stunning real-time weather application. Users can search any city worldwide and instantly see detailed weather information displayed in interactive animated cards. The app demonstrates advanced React patterns, API integration, GSAP-powered animations, and professional UI/UX design.

This project was built as a showcase of frontend development skills including component architecture, state management, animation libraries, and third-party API integration.

2. Key Features

🔍

Real-time Weather Search

Search any city globally and get live data from OpenWeatherMap API instantly.

🎴

ChromaGrid Cards

Interactive mouse-tracking spotlight effect on cards powered by GSAP quickSetter.

✍️

Typewriter Animation

Custom GSAP-powered typewriter component with variable speed and cursor blink.

🌅

Glowing Sun UI

Pulsing sun image with layered CSS box-shadow glow animation synced to heading text.

🔮

Glass Morphism Search

Frosted glass search bar with backdrop-filter blur and focus-triggered glow effect.

📊

Structured Weather Data

3 cards showing condition, temperature, and wind — each with MUI icons and labels.

4. Project Structure

src/
  ├── assets/
    ├── bg.jpg — Dark stormy background (Unsplash)
    ├── sunny.jpg — Sunny card image
    ├── cloudy.jpg — Cloudy card image
    ├── rainy.jpg — Rainy card image
    └── sun.jpg — Glowing sun header image
  ├── components/
    ├── Header/ — Sun image + typewriter heading
    ├── SearchBox/ — City input + API call + state lift
    ├── InfoBox/ — Weather data container
      └── ChromaGrid/ — GSAP interactive cards
    └── TextType/ — Reusable typewriter component
  ├── App.jsx — Root, weatherInfo state, prop drilling
  ├── index.css — Global bg, overlay, layout
  └── main.jsx — React entry point

5. Data Flow Architecture


  User types city → SearchBox
          ↓
  fetch(OpenWeatherMap API)
          ↓
  setWeatherInfo(result) → App.jsx state
          ↓
  <InfoBox info={weatherInfo} />
          ↓
  <ChromaGrid info={info} />
          ↓
  3 structured cards rendered with weather data

6. API Reference

GET https://api.openweathermap.org/data/2.5/weather?q={city}&appid={KEY}&units=metric
main.tempCurrent temperature in °C
main.feels_likeFeels like temperature in °C
main.temp_max / temp_minMax and min temperature
main.humidityHumidity percentage
main.pressureAtmospheric pressure in hPa
weather[0].mainSky condition (Clear, Clouds, Rain...)
weather[0].descriptionDetailed weather description
wind.speedWind speed in m/s

7. Setup & Installation

npm install

npm run dev

Build for Production
